What is this branch?

This is the cucumber-js branch of the colours project.

What does it do?

The cucumber-js branch extends the cucumber branch with step definitions written in JavaScript. These step definitions execute the steps of each Cucumber scenario.

How do I install it?

Ensure that both Node and Node Package Manager (NPM) are installed. Clone the repository and navigate to the local project directory e.g. /home/jimjamz/dev/colours. Install the project dependencies by executing npm install.

How do I use it?

To run the tests, navigate to the /colours directory and execute npm test.

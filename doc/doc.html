What is this branch?

This is the cucumber branch of the colours project.

What does it do?

The cucumber branch contains the Behaviour Driven Development (BDD) aspect of the project, which consists of features presented in the form of use-case stories. These features are comprised of test scenarios that should be designed in line with the requirements of the project.

How do I install it?

Install the Cucumber gem using bundler or gem install. A cucumber plugin for a document viewer / IDE is also preferred for syntax highlighting.

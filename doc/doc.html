What is this branch?

This is the siteprism branch of the colours project.

What does it do?

The siteprism branch contains the same automated tests as the capybara branch, except uses the Page Object Model to abstract away the specific selectors interacted with, making the tests more readable and maintain-able at the same time.

How do I install it?

Install the siteprism gem using bundler or gem install siteprism. A WebDriver such as geckodriver (Firefox) or chromedriver (Chromium) is also required.

How do I use it?

Navigate to the test directory. Ensure all the necessary gems are installed using bundle install. Ensure your WebDriver is running, and run the tests using bundle exec cucumber.

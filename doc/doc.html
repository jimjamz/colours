What is this branch?

This is the nightwatch branch of the colours project.

What does it do?

The nightwatch branch contains the Behaviour Driven Development (BDD) aspect of the project, as well as JavaScript-based automated web browser tests to validate the project functionality.

How do I install it?

Install Nightwatch.js using Node.js and npm with npm install nightwatch.

How do I use it?

Install the package dependencies with npm install and then run the provided tests with npm test.

What is this branch?

This is the capybara branch of the colours project.

What does it do?

The capybara branch contains the automated tests driven by Ruby, but written using the Domain Specific Language (DSL). They tests are governed by the Cucumber Behavioural Driven Development (BDD) scenarios, which are designed in the cucumber branch.

How do I install it?

Install the capybara gem using bundler or gem install capybara. A WebDriver such as geckodriver (Firefox) or chromedriver (Chromium) is also required.

How do I use it?

Navigate to the test directory. Ensure all the necessary gems are installed using bundle install. Ensure your WebDriver is running, and run the tests using bundle exec cucumber.

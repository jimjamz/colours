What is this branch?

This is the rspec branch of the colours project.

What does it do?

The rspec branch contains unit test examples for Ruby classes.

How do I install it?

Install the rspec gem using bundler by navigating to the project directory and running bundle install. Alternatively, install the gem directly by running gem install rspec.

How do I use it?

The RSpec tests are located in the test/spec directory. There are unit tests that exist for the Colour and Palette classes.

To run the tests, navigate to the project's test directory. and run rspec.
